kg-type Web Component Demo

A Web-Component to write out words letter by letter.

Features

  • Write word letter by letter
  • Remove word letter by letter
  • Easily customizable with CSS Custom Properties (aka CSS Variables)
  • Adopts to surrounding CSS because Web Components are awesome
  • No Framework - just JavaScript!

CSS Custom Properties

The following Custom Properties can be set to change the appearance of the component.

    display: var(--kg-type-display, inline-block);
    color: var(--kg-type-color, #000);
    font-size: var(--kg-type-fontsize, 1rem);

Examples

One word:

Code 
                    
<kg-type words="Hello"></kg-type>

Multiple words, type each, clear whole word after:

Code 
                
<kg-type words="Hello, World"></kg-type>

Custom seperator:

Code 
                
<kg-type words="Hello, World!;Hello, Web!;Hello, YOU!" seperator=";"></kg-type>

Custom cursor:

Code 
                
<kg-type words="JavaScript,Node,TypeScript,JSX" cursor="|"></kg-type>

Multiple words, shuffle words:

Code 
                    
<kg-type words="Python, JavaScript, Java, HTML/CSS, JavaScript, Rust, Go, Swift, Shell Scripting, PHP" shuffle="true"></kg-type>

Change font by changing font of wrapper:

Code 
                
<div style="font-family: 'Times New Roman', Times, serif; --kg-type-fontsize: 2rem;">
    <kg-type words="Hello, World"></kg-type>
</div>

Customize color and size with CSS:

Code 
                    
<kg-type style="--kg-type-fontsize: 10rem; --kg-type-display: block; --kg-type-color: blue;" words="Hello, World!; Hello, Web!" seperator=";"></kg-type>

Shuffle words and choose a random word each time the component loads:

Code 
                    
<kg-type words="Python, JavaScript, Java, HTML/CSS, JavaScript, Rust, Go, Swift, Shell Scripting, PHP" shuffle="true"></kg-type>

Delete word letter-by-letter after writing:

Code 
                    
<kg-type words="Python, JavaScript, Java, HTML/CSS, JavaScript, Rust, Go, Swift, Shell Scripting, PHP" delete-reverse="true"></kg-type>
